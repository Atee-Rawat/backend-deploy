# ERP Frontend

This project is built using **React** and serves as the frontend foundation for an ERP application.

## Prerequisites

Before getting started, ensure that you have the following installed:

- **Node.js**: v18.17.0 or later  
- **npm**: v10.8.1 or later (ensure npm is updated)

## Setting Up the Project

Clone this repository to your local machine:

```bash
git clone https://github.com/your-repo/react-project.git

```
 Navigate to the project directory:
```bash
cd rupin_erp_react_js-main
```
Install dependencies:

```bash
npm install
```
Start the development server:

```bash
npm start
```
The project should now be running at http://localhost:3000.
